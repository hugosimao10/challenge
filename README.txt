################### README #####################

1º Abrir a pasta no editor;
2º Abrir um terminal na pasta "Projeto", e escrever:
	-> npm install
	-> npm start
3º Abrir outro terminal na pasta "Projeto", e escrever:
	-> cd client
	-> npm install
	-> npm start
4º Por fim, o browser deverá abrir automaticamente em "localhost:3000".
   Caso não abra automaticamente, abrir o browser e escrever "localhost:3000" (sem as aspas).
